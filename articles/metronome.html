  A basic metronome app
Written 1 August 2024.

Return to All Articles

Over the last few days I've been writing a basic metronome app; this is one of the first major programming projects since when I built this website that I've actually got a working product almost done. I'm pretty proud of it and I'd like to share!

The link to the GitHub repo can be found here. The project is written in pure C, including the one dependency, Raylib (with Raygui). I won't rehash the README of the repo, so I'll just describe it as basically functional. You can set an arbitrary tempo in multiple easy ways, set subdivisions, and change the sound that plays when you click. The program will save your configuration of sounds, and in this configuration you can also set the paths to the Raygui style you want to use, and the directory you want to program to source sounds from.
The biggest problem I've encountered is the text input box for setting the tempo precisely. I think I can solve this problem by separating my data a little better, but for now the text input is janky, though functional once you learn its quirks.
