If you're using my website as a template for yours, I've added a method for creating an image gallery in the article space. At image-gallery-test.html you can see an example of the images by themselves, and below this text I'll show the same thing.



I'm going to work on a script which can generate this column structure automatically because writing it yourself for any handful of files is annoying and stupid.
The basic structure is pretty simple: 
                div id="gallery-row"
                    div id="gallery-column"
                        img ...
                        img ...
                        img ...
                    /div
                    div id="gallery-column"
                        img ...
                        img ...
                        img ...
                    /div
                /div

The layout requires two columns, because the CSS is written manually for two. On my website, two is the best size, but it could be pretty easily changed. Feel free to reach out if you need any help with using my website as a template for your own. This is not intended to be a library or a finished product or anything, but a resource for hobbyists to work from. I'm happy to help as far as my knowledge goes.
